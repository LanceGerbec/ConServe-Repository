# ConServe Project Context

## Tech Stack
- Backend: Node.js + Express + MongoDB + GridFS
- Frontend: React + Vite + Tailwind
- Security: JWT, Helmet, Rate Limiting, Bcrypt

## Key Features Implemented
1. Authentication with role-based access (Student/Faculty/Admin)
2. Research submission with GridFS PDF storage
3. In-app + Email notifications
4. Protected PDF viewer with watermarking
5. Citation generator (APA/MLA/Chicago/Harvard)
6. Analytics dashboard with Recharts
7. Settings management with Cloudinary logo uploads
8. Valid ID verification system
